# 写在前面
本科的时候误入消费主义巨坑，和室友们一起买了许多一百元上下、设计感不错，但一年后就锈迹斑斑甚至直接断掉的项链，后来我痛定思痛鞭策自己疯狂搞钱，给自己搭起了一套购买/制作饰品的完整工作流。我现在只会购买那些传统意义上比较“long-lasting”的、在我消费能力范围内的饰品，包括但不限于银饰（不易腐蚀、易清洗）和串珠（感觉应该是塑料材质蒽），我目前还没有跨入任何金饰、宝石之坑，因为我仍然是穷穷的小女孩一枚。

当然更重要的是谨慎消费，谨慎消费，谨慎消费！虽然我已经看破了消费主义的面纱，但很多时候我还是忍不住为情绪价值花钱...所以一定要谨慎消费奥老师们。

**怎么设计项链？**
我自己做项链的时候是不会画设计图的，可能像姐们这样有无限创意的人就是会直接开做吧嗯嗯。

一般到了想做项链的时候（通常是发现自己没钱买Tiffany的时候，对的），我就会趁着淘宝打折（为什么不用拼多多和1688，因为这上面的选品甚至质量都很难评），在下文会推荐的店铺里完成两个小时的大逛特逛，购入如下基础成分：
* 一些有我喜欢的元素的大小吊坠：火眼金睛之人已能看出...关键词：爱心、镂空、花朵、蝴蝶结
* 一些好看的长的短的链条：具体风格就看个人喜好
* 各式各样的大银扣：用来连结链条，制作完毕后也用来取戴
* 最重要的是足量的开口银环：具体粗细大小要根据自己的需求衡量，一个简单的参考标准：你的吊坠重吗？重就用粗一点的银扣来连接。你的吊坠特别重吗？特别重就只能用大银扣来支撑

最后到货之后就是散落一桌的配件...我们只需根据自己脑中迸发出的灵感拿起链条开始拼接！
![f481c4d1c6fafc164b47d1bfc507670.jpg](/img/user/f481c4d1c6fafc164b47d1bfc507670.jpg)

总结一下，要DIY一条银项链需要付出的：金币、时间、创意，嗯嗯。
